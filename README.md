
# Expense Tracker API

A backend API for tracking personal expenses. Users can sign up, log in, and manage their expenses securely.

## Features

- User registration and login with JWT authentication
- Add, update, delete, and list expenses
- Filter expenses by date (past week, month, 3 months, custom) and category (Groceries, Leisure, Electronics, Utilities, Clothing, Health, Others)

## Tech Stack

- Python, FastApi, SQLAlchemy
- PostgreSQL
- JWT authentication
